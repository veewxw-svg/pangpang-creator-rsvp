# PangPang 博主探店预约小工具本地备用说明

正式网站上线请看：

```text
上线部署说明.md
```

## 每天怎么开

1. 在这台电脑打开终端。
2. 进入 Pang Pang 全局文件夹。
3. 运行：

```bash
node "07_执行交付_当前可发/PangPang_博主探店预约小工具/server.mjs"
```

4. 打开：

```text
http://127.0.0.1:8787/
```

## 日常怎么用

- 新约博主：选“约博主”，贴博主主页链接，写时间人数电话，确认保存。
- 博主发帖后：选“追踪发帖”，贴帖子链接，系统会自动匹配原预约并改成“已发布”。
- 取消预约：在记录里点“取消”，记录会保留，只是状态变成“取消”。
- 查看邮件用 PNG：点右上角打印机图标，或打开：

```text
http://127.0.0.1:8787/api/report.png
```

## 数据放在哪里

正式后台记录会存在：

```text
07_执行交付_当前可发/PangPang_博主探店预约小工具/data/records.json
```

浏览器本地也会留一份备用。

## 下一步接 email

现在已经能生成邮件用 PNG。下一步只需要接一个发件邮箱授权，系统就可以在每次新增、取消、已发布后自动把 PNG 发给营销、店长、老板。
